# Smart Send AI plugin for Claude

Smart Send Plugin for Claude providing detailed shipping capabilities.

## Requirements

This plugin requires an active Smart Send account with access to the Smart Send MCP.

## Installation

See Claude's plugin installation guide [here](https://code.claude.com/docs/en/discover-plugins).

```bash
/plugin marketplace add smartsend/claude-plugin
/plugin install smartsend@smartsend
```

Once installed then Claude will ask for authorization to use the MCP server. Simply login with your Smart Send account and authorize Claude.
